# NextClaw

Homepage: https://nextclaw.io/
Documentation: https://docs.nextclaw.io/
Repository: https://github.com/Peiiii/nextclaw
NPM Package: https://www.npmjs.com/package/nextclaw

Summary:
NextClaw is a local AI workbench and personal operating layer for agents, model providers, message channels, skills, files, automations, and desktop tools.
It helps users start from a task, bring in the right capabilities, and keep work moving inside one workspace.
