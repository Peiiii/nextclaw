# 2026-07-17 · NextClaw v0.25.0

发布时间：2026-07-17
标签：`release` `npm` `minor`

NextClaw v0.25.0 是一次 npm stable minor 更新。本版本让聊天里的结果更像工作台内容，而不是一段段孤立文本：图表、HTML 预览、文件引用、面板应用和工具结果可以更自然地留在同一个会话里。桌面端也会在运行期间继续检查更新，减少“已经发布新版本，但旧应用没有提示”的情况。

![NextClaw 在聊天中直接呈现结构化结果](/release-notes/nextclaw-v0.25.0-inline-results.png)

## 功能

- 聊天中的文件和目录引用支持逐层进入目录和项目全范围搜索，发送后的文件、目录、技能与面板应用标签会以更紧凑的链接样式呈现。
- 新增实验性 UI 注入口。高级用户和社区工具可以在 NextClaw 数据目录放置 `ui-inject.js`，刷新桌面端或浏览器页面后执行自定义界面脚本；删除文件并刷新即可恢复。
- 新增 Skin Studio 技能和 Marketplace 目录能力，方便把可复用的界面皮肤与资源作为技能分发。

## 增强

- HTML 预览不再显示文件名和卡片边框，会根据页面内容调整高度，并在悬停时提供侧栏预览与源码入口。
- Mermaid 图表在历史会话和流式回复里更稳定：打开历史会话时不会先闪现源代码，生成过程中会逐步呈现有效图形，并在更新期间保留上一帧。
- Skill Marketplace 改进最近更新排序、目录刷新、总数表达和历史条目兼容，旧安装类型不再容易拖垮无限滚动末页。
- 桌面端会在应用运行期间定期判断是否需要检查更新，并在窗口重新获得焦点或系统恢复后补查；关闭自动检查时仍不会发起后台请求。

## 修复

- 上下文压缩始终沿用当前会话所选模型；压缩请求失败时不会留下半完成状态，切换到可用模型后可以直接继续会话。
- 窄屏布局从会话列表进入已有会话时，历史消息不再偶尔显示为空；会话详情标题也可以直接下拉搜索和切换会话。
- 聊天中的附件、工作区文件预览和内联内容展开动作更一致，减少内容打开后落到错误表面的情况。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- NPM 安装用户可以继续通过 stable 更新链路，或重新安装 `nextclaw@latest` 获取本版本。
- `ui-inject.js` 是实验性高级能力，不提供安全性、DOM 稳定性或跨版本兼容保证；不需要自定义界面的用户可以忽略它。
- 桌面端后台更新检查仍尊重自动检查开关；关闭后不会因为窗口聚焦或系统恢复而发起后台请求。
