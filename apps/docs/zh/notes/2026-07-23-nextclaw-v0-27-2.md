# 2026-07-23 · NextClaw v0.27.2

发布时间：2026-07-23
标签：`release` `npm` `patch`

NextClaw v0.27.2 是一次 NPM stable patch 更新。回复期间继续发送的消息会按会话排队；切换会话或刷新后，待发消息仍可查看、编辑和删除。技能预览、长会话滚动和 Remote 断线恢复也更稳定。

## 增强

- 会话正在回复时继续发送的消息会按顺序排队。不同会话互不阻塞，切换会话或刷新页面后仍能继续管理各自的待发消息。
- 聊天消息会保留技能名称、来源和文件路径；点击技能可直接打开对应的 `SKILL.md` 预览。

## 修复

- Mermaid 等异步内容改变高度或加载更多历史消息时，向上阅读的位置会保持稳定。
- Remote 连接异常断开后会及时重新连接，不再因历史失败累计而长时间显示 offline。
- 同一台电脑同时运行多个 NextClaw 进程时，Remote 状态会以真正持有连接的服务为准。

## 默认行为与兼容性

- 已安装 `nextclaw@0.27.1` 的用户可以通过 stable 更新通道升级，也可以重新安装 `nextclaw@latest`。
- 本次更新不需要迁移配置、会话或 Remote 域名；默认域名和自定义域名会保持不变。
- 本次 patch 不包含新的桌面安装包。
