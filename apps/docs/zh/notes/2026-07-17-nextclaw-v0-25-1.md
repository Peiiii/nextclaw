# 2026-07-17 · NextClaw v0.25.1

发布时间：2026-07-17
标签：`release` `npm` `patch`

NextClaw v0.25.1 是一次 npm stable patch 更新，重点修复旧版本通过 NPM 安装后看不到更新提示、更新说明入口不够稳定，以及聊天引用菜单和可视化结果生成中的几个细节问题。

## 增强

- 可用更新的版本说明入口会保留在下载或更新按钮的悬停与键盘聚焦区域里；当前运行版本仍可通过版本号查看对应 release note，同时保持顶栏紧凑。
- Agent 的可视化输出指引更清楚：适合展示的结果会优先选择 Markdown、Mermaid、图片或内联 HTML，内联页面完成后只保留可视结果，减少重复文字。

## 修复

- 运行时更新应用后的后续检查会更准确地区分检查失败、下载失败和应用失败；如果新版本已运行但验证失败，页面会展示完整原因和查看日志的命令。
- 聊天引用菜单重新进入“文件与文件夹”时，会从第一行开始高亮，不再恢复到之前某一行。
- 文件与文件夹引用面板在切换目录或搜索结果时保持稳定高度，引用当前文件夹的操作文案也更明确。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- NPM 安装用户可以继续通过 stable 更新链路，或重新安装 `nextclaw@latest` 获取本版本。
- 本次 patch 不包含新的桌面安装包；它面向 NPM runtime 更新链路。
