# 2026-07-18 · NextClaw v0.26.0

发布时间：2026-07-18
标签：`release` `npm` `desktop` `minor`

NextClaw v0.26.0 是一次同时面向 NPM 与桌面端 stable 通道的 minor 更新。长会话浏览、定时任务管理、AI 回复运行信息、移动端新任务入口和内联结果展示都得到改进，并新增内置 Agent Browser。

## 新增能力

- 新的定时任务工作台支持自然语言入口、常用模板、搜索筛选和分页；创建任务后会直接进入对应 AI 会话，任务与会话的关系也更清楚。
- 新增内置 Agent Browser。当已配置的搜索服务不可用时，Agent 仍可以通过浏览器工作流访问网页、检索资料并完成任务。
- 每条 AI 回复现在可以显示实际使用的运行时、模型和 token 用量；消息尾部提供紧凑摘要，详情中保留完整信息。

## 体验增强

- 长会话改用支持动态高度的虚拟时间线，并按需加载更早消息；向上翻阅时会保持阅读位置，HTML、Panel App 和折叠内容展开后也能及时校准高度。
- 运行时身份在助手头像、侧栏和选择器中更加一致；移动端可以从聊天标题栏直接创建新任务，并选择完整的可用运行时列表。
- 会话中的 HTML、Panel App、Mermaid 和表格结果在流式生成结束后保持稳定；工作区导航和设置界面的布局、文案与紧凑模式也更加清楚。

## 修复

- 修复重启后可能因历史检查记录而跳过更新检查的问题。NextClaw 现在每次启动都会立即检查一次，运行期间继续每两小时检查。
- 修复混合回复完成后正文、图片或内联结果偶尔丢失的问题，并改进模型选择面板、聊天输入焦点和任务中止响应。
- 修复远程访问场景中 Panel App 桥接请求缺少必要头信息的问题，让远程内联应用可以正常通信。

## 默认行为与兼容性

- 已安装 `nextclaw@0.25.3` 的用户可以通过 stable 更新通道下载并应用本版本，也可以重新安装 `nextclaw@latest`。
- 无需迁移配置。启动检查和每两小时检查只更新“发现新版本”的状态，不会自动下载或应用更新。
- NextClaw Desktop `0.0.227` 已提供 macOS、Windows 和 Linux 安装包；兼容最低 `0.0.141` 版本要求的现有桌面 launcher，也可以通过 stable 更新通道获取 `0.26.0` runtime bundle。
