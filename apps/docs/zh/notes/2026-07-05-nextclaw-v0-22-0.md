# 2026-07-05 · NextClaw v0.22.0

发布时间：2026-07-05
标签：`release` `chat` `preview` `theme` `runtime`

NextClaw v0.22.0 是一次面向日常使用体验的稳定版更新。这个版本让聊天会话更适合承载连续任务，也让文件、网页、图片和 Panel App 结果更容易在同一个工作区里查看。

## 聊天与任务流

- Agent 运行中可以继续排队输入，当前任务结束后会按顺序发送。
- 长会话里可以用浮动按钮快速回到最新消息。
- 已完成的推理和工具过程会折叠成摘要，最终回答保持可见。
- 失败会话的错误信息会显示在聊天详情页，不只停留在会话列表。
- 宽内容和 inline Panel App 展示更稳定，适合承载更复杂的结果卡片。

## 内容预览与工作区

- `show_file`、`show_url` 和 `show_panel_app` 现在有更明确的展示入口。
- 本地 HTML 可以在右侧工作区里以渲染预览打开，脚本和相对资源更接近真实浏览器行为。
- `show_url` 会以浏览器式标签页打开，带地址、刷新和外部打开入口。
- 文件预览支持语法高亮，面包屑也可以直接浏览附近目录。
- 同一个文件的源码预览和渲染预览可以同时保留在不同标签页。

## 外观与导航

- 新增 Night theme，并改进 Markdown、状态区、错误提示、开关、附件 token、marketplace 详情和 session metadata 的深色可读性。
- SideDock 新增 GitHub 项目入口，也支持隐藏并从 Appearance 设置中恢复。
- DocBrowser 浮动面板支持 header 拖动和四角缩放。
- 当右侧面板过密时，左侧 sidebar 会自动收起，给聊天工作区留出更多空间。
- Sidebar、compact tab 和 Slash 面板的交互细节更一致。

## 附件、图片与运行时

- 修复上传文件、上传图片或粘贴图片后，发送时附件丢失的问题。
- 图片附件不再默认带 `detail: "auto"`，避免部分 OpenAI-compatible provider 拒绝请求。
- 新增 `view_image` agent 工具，模型可以读取本地 PNG、JPEG、WebP 和 GIF 图片。
- NCP 事件增加标准 timing 和 message lifecycle 字段，过程耗时和完成状态更可追踪。
- 工具调用中的取消信号会继续传递到 NCP/MCP 工具执行层。

## 默认行为

- 新配置下 learning loop 默认关闭；已经显式配置过的用户不受影响。

## 升级说明

这是 NPM stable minor 版本。更新后可以通过正常的 NextClaw NPM 安装或更新流程使用本版本。
