# 2026-07-23 · NextClaw v0.27.4

发布时间：2026-07-23
标签：`release` `npm` `patch`

NextClaw v0.27.4 是一次 NPM stable patch 更新。流式回复期间向上阅读时，视图不会再被消息增高重新拉回底部；移动端可以从底部导航直接打开面板应用，带行号打开小型文本文件时也会保留完整内容。

## 增强

- 移动端底部导航新增“应用”入口，可以直接浏览并打开面板应用。

## 修复

- 流式回复期间向上滚动超过 10px 后，会持续保持当前位置；消息继续增高时不会重新夺回滚动位置。
- 带行号打开未超过预览上限的文本或 Markdown 文件时，会显示完整文件并继续定位到目标行，不再错误裁成局部内容。

## 默认行为与兼容性

- 已安装 `nextclaw@0.27.3` 的用户可以通过 stable 更新通道升级，也可以重新安装 `nextclaw@latest`。
- 本次更新不需要迁移配置、会话或文件。
- 本次 patch 不包含新的桌面安装包。
