# 2026-07-15 · NextClaw v0.23.0

发布时间：2026-07-15
标签：`release` `npm` `minor`

NextClaw v0.23.0 是一次 npm stable minor 更新。本版本让 Agent 的图表、内联内容、项目技能和本地文件在同一条会话主线上更稳定地工作，旧会话也更少因为流式更新、聚焦或动态渲染而丢失状态。

![NextClaw 聊天中渲染的 Mermaid 流程图](/release-notes/nextclaw-v0.23.0-mermaid-preview.png)

## 功能

- 聊天消息支持更稳定的流式 Mermaid 图表，Agent 输出中的 Markdown、内联展示和侧栏展示提示也更统一。
- 斜杠选择器、底部技能选择器和 Agent 上下文现在会区分项目、NextClaw、全局与内建技能来源。
- 项目 `.agents/skills` 和项目 `AGENTS.md` 会随会话上下文生效，让项目级技能和规则更自然地参与协作。
- 会话工作区支持关闭概览之外的页面、子会话、草稿和文件标签页，并可从概览重新打开。

## 增强

- Markdown `file:` 链接会保留真实行号，大文件预览会读取目标行附近内容并滚动到对应位置。
- `nextclaw-inline` 文件目标会复用工作台预览能力，覆盖 HTML、Markdown、代码、图片、媒体、PDF 与 Office 文件。
- 用户消息可以复制，最终回复的正文、内联内容和侧栏提示更容易扫读。

## 修复

- 流式输出期间，输入框后台同步不再覆盖页面选区。
- 聚焦输入框连续删除时，光标不再跳到斜杠前。
- 历史 Markdown 和内联 Panel App 在动态 renderer 或工具分组变化时不再反复重新挂载。
- 版本号旁的更新异常图标现在会在悬停或键盘聚焦时展示完整原因、诊断信息和恢复命令。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- NPM 安装用户可以继续通过 stable 更新链路，或重新安装 `nextclaw@latest` 获取本版本。
- 文件预览和 Markdown 渲染能力向后兼容已有消息；新行为优先保持原始文件行号与会话状态连续性。
