# 2026-07-06 · NextClaw v0.22.1

发布时间：2026-07-06
标签：`release` `enhancement` `fix`

NextClaw v0.22.1 是一次 NPM stable patch 更新。本版本重点改进长会话连续性、Agent 管理、取消状态、文件预览和更新前说明，让日常任务更少被状态误报或上下文断点打断。

## 增强

- Agent 管理页现在把“查看详情”和“编辑配置”分开，常用配置事实可以在只读详情中查看，编辑时只保留上下文窗口覆盖项。
- 收起左侧栏后，聊天顶部可以搜索并切换最近会话，不需要先展开侧栏。
- 更新界面可以展示结构化版本说明，用户在下载或应用更新前能先看到本版本变化。
- 本地 HTML 链接默认保留源码预览，只在明确请求渲染视图时打开页面渲染。

## 修复

- 用户主动取消的运行会显示为已取消，不再被当作失败错误展示在会话里。
- 上下文压缩会保留当前用户输入，并把压缩后的上下文放入前置系统提示，长会话滚动后更容易继续原任务。
- 新会话引导内容不会在压缩后的会话里重复插入，减少上下文滚动后的干扰。
- NARP stdio 运行中的 prompt timeout 会按空闲超时处理，正在流式输出的任务不再因为总耗时超过请求超时而失败。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- 这是 NPM stable patch 版本，可通过正常的 NextClaw NPM 安装或更新流程使用。
- 若你的运行环境接入 update manifest，可以使用本版本的结构化 JSON 为更新界面提供说明。
