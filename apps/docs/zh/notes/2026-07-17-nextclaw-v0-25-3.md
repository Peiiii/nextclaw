# 2026-07-17 · NextClaw v0.25.3

发布时间：2026-07-17
标签：`release` `npm` `patch`

NextClaw v0.25.3 是一次 NPM stable patch 更新。通过内置更新功能切换运行时版本后，随 NextClaw 提供的内置技能会继续保留并保持可用。

![NextClaw 使用内置技能在会话中生成 A 股行情图表](/release-notes/nextclaw-v0.25.3-visualize-output.png)

## 修复

- 修复运行时更新包误删内置技能说明和引用资料的问题。应用更新后，NextClaw 仍能发现并使用完整的内置技能，包括 `visualize-output`。
- 首次下载和复用本地更新缓存都会核对内置技能文件；如果文件缺失或内容不一致，更新包构建会直接失败，不再发布不完整的运行时。

## 默认行为与兼容性

- 已安装 `nextclaw@0.25.2` 的用户可以通过 stable 更新通道下载并应用本版本，也可以重新安装 `nextclaw@latest`。
- 无需迁移配置或重新安装技能；本次修复针对 NextClaw 随包提供的内置技能，不改变用户自行安装的技能。
- 本次 patch 不包含新的桌面安装包。
