# 2026-07-13 · NextClaw v0.22.3

发布时间：2026-07-13
标签：`release` `enhancement` `fix`

NextClaw v0.22.3 是一次 NPM stable patch 更新。本版本让聊天中的文件和图片附件更适合在工作区里查看，工具调用过程更清楚，也补齐 Claude Code 和 Codex runtime 在模型选择上的兼容性。

## 功能

- 聊天里的文件附件可以直接打开到右侧工作区预览，不必跳出到新的浏览器窗口。
- 消息图片支持全屏灯箱查看，适合放大检查截图、设计稿和生成图片。
- Claude Code 会话支持 `Runtime default`，可以让 Claude Code 使用自己的配置、鉴权和默认模型。

## 增强

- 工具活动展示更紧凑，推理、工具调用、文件和目录操作会用更清晰的状态行呈现。
- 图片附件的预览按钮、关闭按钮和焦点状态更清楚，键盘操作时不会出现过重的焦点样式。
- cron 卡片、marketplace 搜索、文档浏览器控件和共享表单字段在不同主题下更统一。

## 修复

- Codex runtime 在新的默认模型配置下不再容易出现空响应。
- Marketplace skill 详情页刷新后会恢复文档内容，不再只留下不可用的详情 tab。
- Claude Code runtime 选择显式 NextClaw 模型时仍保持隔离配置，选择 `Runtime default` 时不会覆盖 Claude Code 自己的配置。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- 这是 NPM stable patch 版本，可通过正常的 NextClaw NPM 安装或更新流程使用。
- 桌面端正式版通道不在本次发布范围内；本次只发布 NPM 包。
