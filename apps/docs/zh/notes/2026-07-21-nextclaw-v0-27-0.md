# 2026-07-21 · NextClaw v0.27.0

发布时间：2026-07-21
标签：`release` `npm` `minor`

NextClaw v0.27.0 是一次 NPM stable minor 更新。Remote 访问现在会把设备与本地端口识别为稳定实例，并同时提供固定的随机默认域名和可选的自定义域名；项目接入、长会话上下文管理与 Codex 会话连续性也得到增强。

## 新功能

- Remote 实例不再因服务重启重复出现。同一设备与本地端口会保持同一个实例身份，并继续使用原来的默认域名。
- 每个 Remote 实例都会自动获得一个固定的随机域名；你还可以设置一个全局唯一的自定义域名前缀。两个域名可以同时存在并指向同一实例，设置自定义域名不会覆盖默认域名。
- 可以手动压缩当前会话上下文。支持原生压缩的运行时会直接执行压缩，不支持、会话忙碌或没有可压缩历史时会给出明确反馈。
- 项目列表现在可以直接添加已有目录，不会修改目录内容；Panel App 也可以从标准目录之外的绝对路径打开。

## 体验改进

- Codex 在运行时默认模型与自定义模型之间切换时，会继续沿用原会话上下文，包括启用 reasoning 的 provider。
- Marketplace 技能安装在镜像下载超时时会自动尝试备用来源，并在下载完整后一次性替换，减少半安装状态。
- 工作区文件不存在时，预览区域会显示友好的本地化提示；项目列表间距也更加紧凑。

## 默认行为与兼容性

- Remote 的随机默认域名会自动创建并长期保留；自定义域名遵循全局占用与保留名称规则，释放后才可由其他实例使用。本版本暂不自动回收已占用域名。
- 已有 NPM 安装可以通过 stable 更新通道升级，也可以重新安装 `nextclaw@latest`。
- 本次更新不需要手动迁移配置，不包含新的桌面安装包。
