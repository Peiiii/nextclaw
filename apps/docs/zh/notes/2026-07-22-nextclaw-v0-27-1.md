# 2026-07-22 · NextClaw v0.27.1

发布时间：2026-07-22
标签：`release` `npm` `patch`

NextClaw v0.27.1 是一次 NPM stable patch 更新。当 AI 回复仍在流式输出时，你可以主动向上滚动查看之前的内容，后续消息不会再把视图拉回底部。

## 修复

- 流式回复期间向上滚动超过 10px 后，会稳定退出自动贴底；即使回复继续增长，视图也会停留在当前位置。
- 需要查看最新消息时，仍可使用回到底部按钮重新开启自动贴底。

## 默认行为与兼容性

- 已安装 `nextclaw@0.27.0` 的用户可以通过 stable 更新通道升级，也可以重新安装 `nextclaw@latest`。
- 本次更新不需要迁移配置，也不改变现有会话内容或模型设置。
- 本次 patch 不包含新的桌面安装包。
