# 2026-07-12 · NextClaw v0.22.2

发布时间：2026-07-12
标签：`release` `enhancement` `fix`

NextClaw v0.22.2 是一次 NPM stable patch 更新。本版本让默认界面更清爽，聊天输入和侧边栏操作更集中，工作区文件预览更好用，并改进模型流异常、图片上下文预算和 Panel App 键盘输入。

## 增强

- 默认主题切换为更明亮的工作台样式，侧栏、导航、消息气泡和 marketplace 区域的视觉层次更统一。
- 聊天输入区把模型、thinking、上下文窗口和发送操作收在同一排，模型选择控件不再占用过宽空间。
- 侧边栏支持置顶会话和项目，项目列表可以展开为文件夹行，查找会话和项目更直接。
- 工作区右侧面板支持目录浏览、文件刷新和局部最大化，查看项目材料时不需要离开当前会话。
- 中文和大陆时区用户会优先进入国内文档镜像，仍可通过环境配置回退到全球文档站。

## 修复

- OpenAI Responses 流不完整时会按失败运行处理，不再把半截回答当作成功结果。
- 原生模型流的瞬时失败会带着重试信息和退避策略重试，便于用户和开发者判断真实失败原因。
- 历史图片工具结果不会继续挤占后续模型上下文，高细节图片也会受尺寸和预算控制。
- 聊天设置菜单在流式输出恢复输入焦点时不会误关闭，侧栏搜索图标区域也可以正常聚焦搜索框。
- Panel App iframe 获得焦点后会保留键盘输入，钢琴等交互应用可以直接响应按键。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- 这是 NPM stable patch 版本，可通过正常的 NextClaw NPM 安装或更新流程使用。
- 桌面端正式版通道不在本次发布范围内；本次只发布 NPM 包。
