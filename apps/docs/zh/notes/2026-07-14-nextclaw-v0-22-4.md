# 2026-07-14 · NextClaw v0.22.4

发布时间：2026-07-14
标签：`release` `desktop` `enhancement`

NextClaw v0.22.4 是一次 npm 与桌面端 runtime 的 stable 更新。本版本让聊天工作区更适合日常使用，本地文件预览更统一，也同步刷新了帮助新用户理解产品的对外展示材料。

## 功能

- 聊天工作台新增会话工作区侧栏，文件和任务材料可以更自然地留在当前对话旁边。
- 聊天消息默认使用更扁平的布局，长任务线程更容易扫读。
- 本地文件预览在工作区、附件和消息表面采用更统一的展示方式。

## 增强

- 桌面端 stable 通道已更新到 runtime bundle `0.22.4`，桌面壳版本为 `0.0.220`。
- GitHub README、产品截图和社群入口已同步到当前产品体验。
- GitHub Star 趋势图改为仓库自有静态资产，不再依赖第三方实时图片接口。

## 修复

- 已发布的 `nextclaw` npm 包包含最新 CLI launcher UI 资产。
- 桌面端 stable manifest 与官网下载 fallback 已指向同一个正式桌面版本。

## 默认行为与兼容性

- 本版本不需要手动迁移。
- 桌面端 stable 更新继续兼容 `0.0.141` 及以上 launcher。
- Portable 桌面包仍需手动替换；已安装桌面版可以使用应用内 runtime 更新流程。
