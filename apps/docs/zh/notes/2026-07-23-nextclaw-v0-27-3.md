# 2026-07-23 · NextClaw v0.27.3

发布时间：2026-07-23
标签：`release` `npm` `patch`

NextClaw v0.27.3 是一次 NPM stable patch 更新。Remote 现在会保留断线、重连和心跳诊断信息，短暂的连接器离线页面也会自动重试。下次再遇到 offline，可以根据同一个连接标识对齐本机与 Relay 日志，不再只看到一个无法解释的最终状态。

## 增强

- `nextclaw remote status --json` 会显示连接标识、心跳时间与延迟、断线次数、重连进度和最近一次恢复结果。
- `nextclaw remote doctor --json` 会持续报告当前连接稳定性；连接恢复后，最近一次异常证据不会立即消失。

## 修复

- Relay 会确认心跳并记录连接、关闭和不可用事件，便于区分本地服务、网络传输和云端连接层的问题。
- 域名访问遇到短暂的 Remote connector offline 时会展示事件标识并自动重试；会话过期或被撤销仍会直接提示重新登录。
- 心跳消息不再占用普通 Remote 请求配额。

## 默认行为与兼容性

- 已安装 `nextclaw@0.27.2` 的用户可以通过 stable 更新通道升级，也可以重新安装 `nextclaw@latest`。
- 本次更新不需要迁移配置、实例或域名；默认域名和自定义域名都会保持不变。
- 本次 patch 不包含新的桌面安装包。
