# 2026-07-20 · NextClaw v0.26.1

发布时间：2026-07-20
标签：`release` `npm` `patch`

NextClaw v0.26.1 是一次 NPM stable patch 更新。聊天消息的运行信息更简洁，移动端输入工具栏更紧凑，定时任务模板也会根据窗口宽度自动调整。

## 修复

- AI 回复没有 token 用量时，不再显示不可用占位；完整运行信息仍可从消息操作中查看，弹窗打开和关闭时也不会跳位。
- 移动端聊天输入面板在空间足够时会保持工具栏控件单行排列，减少不必要的面板高度。
- 定时任务模板会根据可用内容宽度调整列数，在窄窗口和移动端保持完整可读。

## 默认行为与兼容性

- 已安装 `nextclaw@0.26.0` 的用户可以通过 stable 更新通道升级，也可以重新安装 `nextclaw@latest`。
- 本次更新不需要迁移配置，也不改变现有会话、定时任务或模型设置。
- 本次 patch 不包含新的桌面安装包。
