# 2026-07-18 · NextClaw v0.25.4

发布时间：2026-07-18
标签：`release` `npm` `patch`

NextClaw v0.25.4 是一次 NPM stable patch 更新。通过远程访问打开会话中的小应用时，应用现在可以正常读取数据并执行操作；Claude Code 等 stdio Agent 的慢响应可以更快终止；设置导航和 Agent 运行时辨识也更加清楚。

## 修复

- 修复远程访问中的小应用显示“加载失败”、无法读取数据或执行操作的问题。
- 上游响应缓慢或不可用时，终止当前回复不再长时间等待；终止后仍可继续使用同一会话。
- 修复重复点击当前模型输入框时选项面板意外关闭的问题。

## 体验改进

- 设置侧栏的密度、底部顺序和链接行为保持一致；主题与语言偏好统一放入“外观”。
- 助手消息头像和会话侧栏会更清楚地呈现当前 Agent 运行时，运行时选择器的图标与提示也保持一致。
- NextClaw 每次启动后都会立即检查一次可用更新，运行期间继续按固定周期检查，不会自动下载或应用更新。

## 默认行为与兼容性

- 已安装 `nextclaw@0.25.3` 的用户可以通过 stable 更新通道下载并应用本版本，也可以重新安装 `nextclaw@latest`。
- 无需迁移配置；升级并重启运行时后，远程小应用和终止响应修复即可生效。
- 本次 patch 不包含新的桌面安装包。
