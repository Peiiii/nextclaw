# 2026-07-17 · NextClaw v0.25.2

发布时间：2026-07-17
标签：`release` `npm` `patch`

NextClaw v0.25.2 是一次 NPM stable patch 更新。通过 NPM 安装且持续运行的 NextClaw 现在会定期检查新版本，但发现更新后只提示，不会自行下载或应用。

## 增强

- 通过 NPM 安装的 NextClaw 会按最近一次检查时间安排后续检查，并在持续运行期间每两小时再次检查，不再需要通过重启或手动刷新才能发现新版本。
- 发现新版本后，NextClaw 会停在“有可用更新”状态；只有用户明确点击下载和应用后，才会消耗网络、磁盘并切换运行版本。

## 修复

- 切换 stable 或 beta 更新通道时，会先等待旧通道的检查结束，再查询新通道，避免继续展示旧通道的结果。

## 默认行为与兼容性

- 自动检查更新现在是固定行为；原有“关闭自动检查”和“自动下载”偏好不再提供，旧偏好会被安全忽略，无需手动迁移。
- NPM 安装用户可以继续使用 stable 更新链路，或重新安装 `nextclaw@latest` 获取本版本。
- 本次 patch 不包含新的桌面安装包；桌面安装版会在后续桌面发布中获得同一套更新行为。
