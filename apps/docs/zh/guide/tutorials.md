## 教程、指南和手册的区别

教程回答“我照着哪条路径能跑通一个经典场景”。  
指南回答“我怎么完成这件事”。  
手册回答“这个能力是什么、有哪些规则、边界是什么”。

如果你只想先跑通，优先看教程。  
如果你要理解能力怎么选，去看 [指南](/zh/guide/model-selection)。  
如果你要查正式说明，去 [配置手册](/zh/guide/configuration) 或 [运行与托管手册](/zh/guide/runtime-hosting)。
