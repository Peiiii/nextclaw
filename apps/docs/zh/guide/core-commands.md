# 核心命令

这页只放普通用户最常用的少量命令。完整命令请看 [命令索引](/zh/guide/commands)。

## 启动与停止

```bash
nextclaw start
nextclaw restart
nextclaw stop
```

## 状态与诊断

```bash
nextclaw status
nextclaw doctor
```

## 版本与更新

```bash
nextclaw --version
nextclaw update
```

## 自启动检查

```bash
nextclaw service autostart status
nextclaw service autostart doctor
```

## 远程访问检查

```bash
nextclaw remote status
nextclaw remote doctor
```

## 使用原则

第一次上手只需要 `start`、`status`、`doctor`、`stop`。  
其他命令只有在对应场景需要时再查。
