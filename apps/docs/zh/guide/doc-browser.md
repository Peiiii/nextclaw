# Doc Browser

Doc Browser 是 NextClaw 的全局右侧浏览区。它可以在主任务旁打开网页、文档和多个标签，适合边查资料边继续对话，也可以停靠或浮动使用。

![技能市场旁打开的文档浏览器](/product-screenshots/nextclaw-skills-doc-browser-cn.png)

## 它和会话工作区的区别

- **会话工作区**围绕当前任务的本地文件、结果、子任务和定时任务。
- **Doc Browser**是跨页面保留的全局浏览区，用来查看网页、文档、GitHub 或其他参考内容。

## 常见用法

- 查看一个 Skill 或 MCP 的说明，同时在主界面安装和测试。
- 打开参考网页，在会话里要求 Agent 对比或整理。
- 查看 GitHub 仓库、产品文档或本地服务页面。
- 同时保留多个浏览标签，在不同任务页面之间切换。

Doc Browser 便于查看内容，但网页是否被 Agent 自动读取、操作或发送给模型，仍取决于当前任务使用的工具和你给出的指令。
