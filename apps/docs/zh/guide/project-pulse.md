# Project Pulse

`Project Pulse` 是 NextClaw 面向公开文档站的产品脉搏页。

它把工程变化和产品变化收敛到同一个页面里：代码体积趋势、commit 节奏、release 批次、近期产品笔记，以及当前自动化截图。
