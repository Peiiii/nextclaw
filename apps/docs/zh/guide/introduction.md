# NextClaw 是什么

NextClaw 是一个运行在你自己机器上的个人 AI 操作层。它不是单纯的聊天窗口，而是把模型、渠道、自动化任务和宿主机器能力组织到一个入口里。

你可以从最小配置开始：安装 CLI，启动服务，打开浏览器界面，配置一个模型提供方，然后得到第一条可用回复。之后再逐步接入聊天渠道、定时任务、远程访问和长期运行方式。

## 它解决什么问题

- 你不想在多个模型、多个工具、多个聊天入口之间来回切换。
- 你希望 AI 不只回答问题，还能通过渠道、任务和本机能力持续帮你做事。
- 你希望先在自己的机器上跑通，再决定是否长期托管、远程访问或接入更多能力。

## 它不是什么

- 它不是只给开发者看的 SDK 文档。
- 它不是必须先学完整命令体系才能用的工具。
- 它也不是把所有功能硬塞进一个界面里的万能杂货箱。
