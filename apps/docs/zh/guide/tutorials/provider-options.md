# 先选接入方式

第一次配置模型时，先选一条最容易成功的路。不要一开始就比较所有 provider。

## 推荐选择

| 你的情况 | 建议路径 |
|----------|----------|
| 想最快体验 | 选择已有账号或最容易获得的 provider |
| 已经有 API Key | 直接配置对应 provider |
| 想本地运行 | 使用 Ollama 路径 |
| 想接 Claude Code / Codex / Hermes | 看专门集成教程 |

## 最小判断

你只需要确认三件事：

- 是否能获得认证信息
- 是否知道要用哪个模型
- 是否能在 UI 里得到一次正常回复
