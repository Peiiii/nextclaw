# Linux 桌面安装

这页用于 Linux 桌面版安装。  
如果你使用的是 NPM CLI 安装，请回到 [快速开始](/zh/guide/getting-started)。

## 适合谁

- 你希望用桌面安装包
- 你使用 Debian / Ubuntu 系发行版
- 你希望通过系统包管理器安装和更新

## 安装后要做什么

1. 启动 NextClaw。
2. 打开本地 UI。
3. 配置模型提供方。
4. 发送第一条消息。

## 和 NPM 安装的区别

NPM 安装更偏 CLI 路径。  
桌面安装更偏系统应用路径。

无论哪种路径，最终都应该完成：

- UI 可打开
- provider 可用
- 能得到正常回复
