# 汇总用户反馈并排出问题优先级

把工单、问卷、群聊记录、访谈笔记或应用商店评论交给 NextClaw，合并重复问题，统计频次，并把每个结论连回原始证据。

## 准备材料

- 导出的 CSV、Excel、文本或消息记录；
- 时间范围、产品版本和用户类型；
- 已知分类或团队当前关注的问题；
- 隐私处理要求；
- 希望得到的表格、报告或看板格式。

处理包含姓名、联系方式或业务敏感信息的材料前，先制作脱敏副本，并确认使用的模型和工具是否允许接收这些数据。

## Agent 会做什么

1. 检查字段、重复记录和隐私数据。
2. 给出初步分类并让你确认粒度。
3. 聚类、统计频次和受影响范围。
4. 为结论保留匿名证据和来源行号。
5. 生成表格、摘要和可视化报告。

## 检查结果

- 两个不同问题是否被错误合并。
- 高频是否真的等于高优先级，还是只更容易被反馈。
- 原文抽样能否支持摘要结论。
- 版本、用户类型和严重程度是否被保留。
- 脱敏结果中是否仍包含可识别信息。

## 继续使用

把稳定分类、优先级公式和报告格式保存为 Skill；需要每周更新时，再接入[定时任务](/zh/guide/cron)。
