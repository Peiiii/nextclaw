# Release Notes

If you want to know "what changed recently that is worth my attention or upgrade," this is the best place to start.

This page points to public, user-facing updates rather than raw implementation logs. It focuses on:

- why the change matters
- how you can use it
- where to go if you want more detail

## Entry Point

- [Product Notes](/en/notes/)

## How this differs from the other project pages

- `Release Notes`: short updates that lead with user value
- [Project Pulse](/en/project/project-pulse): the overall public shipping dashboard
- [Roadmap](/en/project/roadmap): what is being prioritized next
- [Blog](/en/blog/): longer essays about judgment, trade-offs, and design
