# 2026-07-22 · NextClaw v0.27.1

Published: 2026-07-22
Tags: `release` `npm` `patch`

NextClaw v0.27.1 is an npm stable patch release. While an AI reply is still streaming, you can scroll up to read earlier content without later updates pulling the view back to the bottom.

## Fixes

- Scrolling more than 10px away from the bottom during a streaming reply now reliably exits automatic bottom-following, even as the reply continues to grow.
- When you want to return to the latest message, the existing scroll-to-bottom button re-enables automatic bottom-following.

## Defaults and Compatibility

- Existing `nextclaw@0.27.0` installations can update through the stable channel or reinstall `nextclaw@latest`.
- No configuration migration is required, and existing conversation content and model settings are unchanged.
- This patch does not include a new desktop installer.
