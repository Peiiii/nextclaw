# 2026-07-23 · NextClaw v0.27.4

Published: 2026-07-23
Tags: `release` `npm` `patch`

NextClaw v0.27.4 is an npm stable patch release. Reading upward during a streaming reply no longer gets pulled back to the bottom as the message grows. Mobile users can open panel apps from the bottom navigation, and opening a small text file at a specific line now keeps the complete file visible.

## Enhancements

- Mobile bottom navigation now includes an Apps entry for browsing and opening panel apps directly.

## Fixes

- After scrolling more than 10 px away from the bottom during a streaming reply, the reading position remains escaped as the active message grows.
- Opening a text or Markdown file at a specific line keeps the complete file when it is below the preview limit, while still focusing the requested line.

## Defaults and Compatibility

- Existing `nextclaw@0.27.3` installations can update through the stable channel or reinstall `nextclaw@latest`.
- No configuration, session, or file migration is required.
- This patch does not include a new desktop installer.
