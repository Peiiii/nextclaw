# 2026-07-18 · NextClaw v0.25.4

Published: 2026-07-18
Tags: `release` `npm` `patch`

NextClaw v0.25.4 is an npm stable patch release. Panel apps opened through remote access can now load data and perform actions normally, and Agent runtimes are easier to identify throughout chat.

## Fixes

- Fixed panel apps showing a loading failure or being unable to read data and perform actions through remote access.
- Fixed model option panels closing when users click an already-focused model input.

## Experience Improvements

- Assistant avatars and session sidebar entries identify the active Agent runtime more clearly, with consistent runtime selector icons and hints.
- NextClaw checks for an available update immediately after every start and continues checking periodically while running. It does not download or apply updates automatically.

## Defaults and Compatibility

- Existing `nextclaw@0.25.3` installations can download and apply this release from the stable channel, or reinstall `nextclaw@latest`.
- No configuration migration is required. Restart the upgraded runtime to activate the remote panel app fix.
- This patch does not include a new desktop installer.
