# 2026-07-17 · NextClaw v0.25.3

Published: 2026-07-17
Tags: `release` `npm` `patch`

NextClaw v0.25.3 is an npm stable patch release. Built-in skills shipped with NextClaw now remain present and available after applying a runtime update.

## Fixes

- Fixed runtime update bundles incorrectly removing built-in skill instructions and referenced assets. Updated runtimes can still discover and use the complete built-in skill set, including `visualize-output`.
- Fresh deployments and restored update caches now verify built-in skill files. A runtime bundle fails during construction if any skill file is missing or changed instead of publishing an incomplete update.

## Defaults and Compatibility

- Existing `nextclaw@0.25.2` installations can download and apply this release from the stable channel, or reinstall `nextclaw@latest`.
- No configuration migration or skill reinstallation is required. The fix covers skills bundled with NextClaw and does not change user-installed skills.
- This patch does not include a new desktop installer.
