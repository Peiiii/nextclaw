# 2026-07-14 · NextClaw v0.22.4

Published: 2026-07-14
Tags: `release` `desktop` `enhancement`

NextClaw v0.22.4 is a stable update for npm and the desktop runtime. It makes the chat workspace easier to use day to day, brings local file previews into a more consistent shape, and refreshes the public product materials that help new users understand what NextClaw does.

## Features

- The chat workspace has a session workspace sidebar, so files and task materials stay closer to the conversation.
- Chat messages use a flatter default layout, making long task threads easier to scan.
- Local file previews are more consistent across workspace, attachment, and message surfaces.

## Enhancements

- The desktop stable channel now ships runtime bundle `0.22.4` with desktop shell `0.0.220`.
- GitHub README, product screenshots, and community entry points now match the current product experience.
- The project star history chart is served from a repository-owned static asset instead of a third-party live image.

## Fixes

- The published `nextclaw` npm package now includes the refreshed UI assets used by the CLI launcher.
- Desktop stable manifests and website download fallbacks now point to the same stable desktop release.

## Defaults and Compatibility

- No manual migration is required.
- Stable desktop updates remain compatible with launcher `0.0.141` and newer.
- Portable desktop builds still require manual replacement; installed desktop builds can use the in-app runtime update flow.
