# 2026-07-13 · NextClaw v0.22.3

Published: 2026-07-13
Tags: `release` `enhancement` `fix`

NextClaw v0.22.3 is an NPM stable patch release. This update makes chat file and image attachments easier to inspect inside the workspace, clarifies tool activity, and improves model-selection compatibility for Claude Code and Codex runtimes.

## Features

- Chat file attachments can open directly in the workspace side preview instead of a new browser window.
- Message images can open in a fullscreen lightbox for inspecting screenshots, designs, and generated images.
- Claude Code sessions support `Runtime default`, allowing Claude Code to use its own configuration, authentication, and default model.

## Enhancements

- Tool activity rows are more compact, with clearer status for reasoning, tool calls, file work, and directory work.
- Image attachment preview controls, close controls, and focus states are easier to read without heavy focus rings.
- Cron cards, marketplace search, document browser controls, and shared form fields are more consistent across themes.

## Fixes

- Codex runtime no longer tends to return empty responses with the newer runtime default model setup.
- Marketplace skill detail pages restore their document content after browser refreshes instead of leaving the detail tab unavailable.
- Claude Code runtime keeps isolated configuration for explicit NextClaw model choices, while `Runtime default` no longer overwrites Claude Code's own configuration.

## Defaults and Compatibility

- No manual migration is required.
- This is an NPM stable patch release available through the normal NextClaw NPM install or update flow.
- The desktop stable channel is outside this release scope; this batch publishes NPM packages only.
