# 2026-07-17 · NextClaw v0.25.1

Published: 2026-07-17
Tags: `release` `npm` `patch`

NextClaw v0.25.1 is an npm stable patch update focused on older npm-installed versions seeing the right update prompt, keeping release-note links discoverable, and tightening chat reference plus visual-result behavior.

## Enhancements

- Available-update release notes stay inside the download or update control's hover and keyboard-focus surface. The current running version still links from the version label, keeping the header compact.
- Agent visual-output guidance is clearer: suitable results can choose Markdown, Mermaid, images, or inline HTML, and completed inline pages keep the visual result without repeating extra surrounding text.

## Fixes

- Runtime update follow-up checks distinguish check, download, and apply failures more accurately. If the new version is already running but verification fails, the UI shows the full reason and the log command.
- Re-entering "Files and folders" from the chat reference menu starts highlight from the first row instead of restoring a previous row.
- The file and folder reference panel keeps a stable height while switching directories or search results, and the action for referencing the current folder is clearer.

## Defaults and Compatibility

- No manual migration is required.
- NPM users can continue through the stable update path or reinstall `nextclaw@latest` to get this release.
- This patch does not include a new desktop installer; it targets the NPM runtime update path.
