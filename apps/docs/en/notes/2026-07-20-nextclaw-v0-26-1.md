# 2026-07-20 · NextClaw v0.26.1

Published: 2026-07-20
Tags: `release` `npm` `patch`

NextClaw v0.26.1 is an npm stable patch release. Chat runtime details are cleaner, the mobile composer toolbar uses space more efficiently, and scheduled task templates now adapt to the available width.

## Fixes

- AI replies no longer show an unavailable placeholder when token usage is missing. Full runtime details remain available from message actions, and the dialog no longer shifts while opening or closing.
- The mobile chat composer keeps toolbar controls on one row when space allows, reducing unnecessary panel height.
- Scheduled task templates adjust their column count to the available width and remain readable in narrow windows and on mobile devices.

## Defaults and Compatibility

- Existing `nextclaw@0.26.0` installations can update through the stable channel or reinstall `nextclaw@latest`.
- No configuration migration is required, and existing conversations, scheduled tasks, and model settings are unchanged.
- This patch does not include a new desktop installer.
