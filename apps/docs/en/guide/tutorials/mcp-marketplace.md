# MCP Tutorial

MCP is for connecting external capabilities to NextClaw. It is an extension path, not a first-run requirement.

## When you need it

- you already have an MCP server
- you want NextClaw to call external system capabilities
- you need to connect an existing toolchain into an AI workflow

## Steps

1. Confirm the MCP server works by itself.
2. Add the integration configuration in NextClaw.
3. Save and reload configuration.
4. Verify with one low-risk task.

## Usage principles

- Prove the MCP server works first.
- Then connect it to NextClaw.
- Do not base first onboarding on a complex MCP chain.
