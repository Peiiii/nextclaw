# Feishu Setup

Feishu is a good path for bringing NextClaw into a team collaboration surface. For the first setup, aim for the minimum working connection.

## Before setup

- NextClaw is already running
- a model provider works
- you have the Feishu app or permissions required for integration

## Steps

1. Prepare the Feishu app information and credentials.
2. Add the Feishu channel in NextClaw.
3. Save the configuration.
4. Run `nextclaw channels status`.
5. Send one test message from Feishu.

## Common issues

- insufficient permission scope
- wrong token or app secret
- callback URL is unreachable
- service is not staying online
