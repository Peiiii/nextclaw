# Qwen Portal Setup

This is one provider setup path. The goal is not to compare every model; it is to get Qwen Portal working.

## Steps

1. Prepare a usable account or authentication details.
2. Add a provider in the NextClaw UI.
3. Enter the connection details.
4. Select a default model.
5. Save and send one real message.

## Verify

```bash
nextclaw doctor
```

If the UI returns a normal reply, this provider path works.

## If it fails

- account is usable
- model name is correct
- network can reach the platform
- default model was saved
