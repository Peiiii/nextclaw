# Task guides

These are concrete jobs, not feature labels. Each guide explains the source material, a prompt you can adapt, the agent's key steps, the deliverable, and how to review it.

## Work with information

### [Collect data and build a chart report](/en/tasks/data-analysis)

Read websites, CSV, or spreadsheets; clean the data; confirm metrics; and deliver charts, conclusions, and an HTML or Markdown report.

### [Organize, rename, and summarize local files](/en/tasks/file-processing)

Inventory a directory and preview the plan before classifying, renaming, extracting, or summarizing its files.

### [Research multiple sources with citations](/en/tasks/research-writing)

Collect web and local material, compare evidence, and turn facts, disagreements, and references into a research document.

### [Draft an article or report from source material](/en/tasks/writing)

Turn interviews, notes, previous drafts, and references into an article, weekly report, proposal, or release note.

### [Analyze customer feedback and prioritize issues](/en/tasks/feedback-analysis)

Combine tickets, conversations, or surveys, merge duplicate issues, count patterns, and preserve evidence for each conclusion.

## Create and keep work running

### [Generate an image and keep the local file](/en/tasks/image-creation)

Define the use, composition, and dimensions, generate the image, save it locally, and refine it in the same task.

### [Build a local app beside the task](/en/tasks/build-local-app)

Turn repeated work into an HTML page, script, or Panel App and review it in the right side.

### [Inspect and modify a codebase](/en/tasks/code-project)

Understand the repository, reproduce a problem, change the code, run checks, and inspect the real result.

### [Generate and send a scheduled brief](/en/tasks/scheduled-brief)

Prove the sources and output manually before choosing the schedule, working directory, and destination channel.

### [Handle requests from Weixin or Feishu](/en/tasks/chat-channel-work)

Send a request from a familiar conversation, let the local agent use its files and tools, then return the result.

For a first task, choose something you can verify in a few minutes and state where the result should be saved. Start with [Install NextClaw](/en/guide/install) and [Create your first task](/en/guide/create-task).
