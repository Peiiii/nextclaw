# Starter Card

`starter-card` 是一个最小官方示例 app，专门用于验证本地安装、宿主 bridge 和 UI 展示链路。

## Local workflow

```bash
napp inspect ./apps/examples/starter-card
napp pack ./apps/examples/starter-card
napp install ./apps/examples/starter-card
napp run nextclaw.starter-card
```

## Publish workflow

```bash
napp publish ./apps/examples/starter-card
```
