English Version

NextClaw Desktop 0.0.214 stable release for runtime 0.21.11.

- Promotes the verified 0.21.11 runtime bundle to the stable desktop channel.
- Includes macOS, Windows, and Linux desktop installers, Windows portable packages, update bundles, stable update manifests, and Linux APT publishing.
- Carries the refreshed minimalist NextClaw UI theme system and packaged UI assets from the 0.21.11 runtime line.
