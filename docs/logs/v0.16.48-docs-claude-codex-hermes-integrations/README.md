# v0.16.48-docs-claude-codex-hermes-integrations

## 迭代完成说明（改了什么）

本次补齐了面向普通用户的 `Claude Code / Codex / Hermes` 集成文档，并正式接入文档站导航。

- 新增中英文教程页：
  - `apps/docs/zh/guide/tutorials/claude-codex-hermes.md`
  - `apps/docs/en/guide/tutorials/claude-codex-hermes.md`
- 在中英文 `教程总览 / Tutorials` 页面加入入口，避免用户只能靠搜索定位
- 在中英文 `快速开始 / Quick Start` 页加入直达链接，让新用户在首次配置阶段就能发现这三条集成路径
- 在 VitePress 中英文 sidebar 的 `学习与资源 / Learn & Resources` 中加入正式导航项
- 把中英文 `Skills` 教程一起改成 UI-first，用 `Marketplace -> Skills -> Install -> 回聊天勾选 Skills` 的普通用户路径替换原先偏命令行的写法

本次文档内容的产品侧重点是：

- 先帮用户判断三条集成分别适合谁
- 再给出最短启用路径
- 最后给出统一的最小验收方式，降低排错成本

本轮还对刚写出的第一版思路做了纠偏：

- 删除了不适合普通用户的命令行安装、手动 runtime 配置、手写 `config.json` 等内容
- `Claude / Codex` 收敛为纯 UI 插件安装路径
- `Hermes` 收敛为 `Marketplace -> Skills` 安装 `Hermes Runtime`，再在聊天里使用 skill 的路径
- 同步把原本偏命令行心智的 `Skills` 教程改成普通用户可直接照着点的 UI 路径
