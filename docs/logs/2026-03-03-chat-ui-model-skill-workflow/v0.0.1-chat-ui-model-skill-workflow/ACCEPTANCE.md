# 用户/产品验收步骤

## 验收环境
- 打开 Web UI：`/chat`
- 准备至少一个已配置 Provider（确保有可选模型）
- 安装至少一个 skill（可见描述）

## 验收清单

1. 模型切换
- 打开对话输入区底部选择器。
- 预期：显示为模型选择，不再是 Agent 选择。
- 预期：当前值展示为单行 `Provider/model`。
- 预期：下拉列表仅出现已配置 Provider 的模型。

2. Skill 描述与语言
- 打开技能弹层。
- 预期：每条 skill 有描述。
- 将界面语言切到中文，再切回英文。
- 预期：描述随界面语言切换（有对应语言时优先使用；无对应语言时回退默认描述）。

3. 工具调用工作流视觉
- 发起一个会触发多个工具调用的任务。
- 预期：工具卡片按步骤串联显示（有线、有编号）。
- 预期：工具输出仍可查看/展开。

4. 对话宽度
- 在桌面端查看会话区与输入区。
- 预期：内容宽度明显比旧版本更宽，阅读密度更合理。

5. 回归
- 普通消息发送/接收正常。
- 新建会话、删除会话正常。
