# v0.0.2 User Acceptance

## 用户/产品视角验收步骤

1. 打开聊天页，点击“技能”按钮，搜索并勾选两个技能（例如 `bird`、`pdf`）。
2. 观察输入框区域：已选技能以标签形式显示。
3. 发送消息后，确认：
   - 消息正常发送；
   - 输入框中的技能标签被清空；
   - 下次发送默认不携带上次技能。
4. 触发一个会连续调用多个工具的问题（如“检索并读取多个文件后总结”）。
5. 观察工具展示：
   - 顶部出现 `Tool1 → Tool2 → Tool3` 风格工作流摘要；
   - 默认不展开细节；
   - 展开后可看到每步参数/结果。
6. 切换界面语言，确认技能描述随语言切换（中文界面优先 `descriptionZh`）。
7. 检查模型下拉：仅显示已配置 provider 的模型，当前选中展示为单行 `Provider/model`。
